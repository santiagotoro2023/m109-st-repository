System aktualisieren

Befehl im Terminal:

sudo apt update && sudo apt upgrade -y

Git installieren

Befehl im Terminal:

sudo apt install git-all

Projektordner erstellen

Ordnername: m109-st-repo

mkdir m109-st-repo

GitHub Repository

Online erstellt über die GitHub-WebGUI mit dem gleichen Namen.

Remote verbinden

GitHub-Repository mit lokalem Ordner verbinden:

git remote add origin https://github.com/santiagotoro2023/m109-repo-st.git

HTML-Datei erstellen

Eine index.html-Datei wurde im Projektordner angelegt.

Verbindung via SSH
